## ğŸ”§ Customization

### Add Your Own Prompts
Edit the JSON in the web interface:
```json
[
  {
    "number": 1,
    "prompt": "Your custom prompt here"
  },
  {
    "number": 2,
    "prompt": "Another prompt"
  }
]
```

### Change Image Settings
In `app.py`, modify the `generate_in_background()` function:
```python
output = replicate.run(
    model_name,
    input={
        "prompt": enhanced_prompt,
        "num_outputs": 1,
        "aspect_ratio": "1:1",      # Change to "4:5" for portrait
        "output_format": "jpg",      # or "png"
        "output_quality": 90         # 1-100
    }
)
```

### Add More Models
In `app.py`, add to `model_map`:
```python
model_map = {
    'flux-schnell': 'black-forest-labs/flux-schnell',
    'your-model': 'owner/model-name'
}
```

Then update the HTML dropdown in `index.html`.

## ğŸ› Troubleshooting

### "Invalid API token" error
- Make sure token starts with `r8_`
- Copy entire token without spaces
- Check [replicate.com/account](https://replicate.com/account) for valid token

### Images not generating
- Check you have credits on Replicate
- Try FLUX Schnell (free tier)
- Check Replit logs for errors

### Slow generation
- Normal: 20-40 seconds per image
- 30 images = 10-15 minutes total
- Don't close browser during generation

### Out of memory
- Replit free tier has limited RAM
- Clear old images: Click "Clear All"
- Generate in smaller batches (10 at a time)
